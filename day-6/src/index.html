Dev Reference Guide

Quick reference for core web development keywords

HTML Document Structure

HTML

Boilerplate with <!DOCTYPE html>, <html>, <head>, <body>.

Headings

HTML

<h1> to <h6> define headings of different importance.

Paragraphs

HTML

<p> defines a paragraph of text.

Hyperlinks

HTML

<a> creates links to other pages or resources.

Lists

HTML

<ul> for unordered, <ol> for ordered, <li> for list items.

Images

HTML

<img> displays an image in the document.

Buttons

HTML

<button> creates a clickable button.

Forms

HTML

Used to collect user input; includes <input>, <textarea>, <select>.

Tables

HTML

<table>, <tr>, <td> are used for tabular data.

Color

CSS

Sets text color using names, hex, RGB, or HSL values.

Background

CSS

Sets background color or image of an element.

Margin

CSS

Controls outer spacing of an element.

Padding

CSS

Controls inner spacing of an element.

Border

CSS

Sets width, style, and color of element borders.

Display

CSS

Defines display behavior: block, inline, flex, or grid.

Flex

CSS

Flexible layout for children in a container.

Font

CSS

font-size sets text size; font-family sets typeface.

let / const / var

JavaScript

Variable declarations: block-scoped (let/const) or function-scoped (var).

Functions

JavaScript

Reusable blocks of code to perform tasks.

If / Else

JavaScript

Conditional statements to control code flow.

Loops

JavaScript

for and while loops to iterate over code blocks.

DOM

JavaScript

Document Object Model representing page structure.

Events

JavaScript

Actions like clicks or inputs that trigger JavaScript code.

Return

JavaScript

Returns a value from a function.

VS Code UI

VS Code

Sync extensions, themes, and settings across devices.

Live Server

ESLint

Extension

Identifies and fixes JavaScript problems.

Keyboard Shortcuts

Extension

Customizable shortcuts for efficiency.

Tips & Tricks

VS Code

Productivity features like multi-cursor, split view, and snippets.
